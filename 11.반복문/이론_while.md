# 11-2.while

## While 문
    - format
        while(조건) {
            내용
        }
    - 조건을 만족하는 동안 본문(내용)을 반복

## Do/While 문
    - format
        do {
            내용
        } while(조건)
    - 먼저 do를 한번 실행하고 조건을 만족하는 동안 계속 반복

### 주의점
    - While문은 조건을 만족하는 동안 본문이 반복 실행
    - 무조건 본문에 While문의 조건을 만족하지 못하도록 하는 내용이 있어야 함
